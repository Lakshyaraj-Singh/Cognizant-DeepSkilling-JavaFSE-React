# Exercise 2: E-commerce Platform Search Function

## Explain Big O notation and how it helps in analyzing algorithms

=> Big O notation is used to compare time complexity of an algorithm.The amount of time algorithm takes for an inpput of size n.Compares different algorithms and help us choose the suitable algorithm for a given problem.

### For Linear search

**Best Case = O(1).**

* Found at first position.

**Average Case: O(n)**

* The product is found somewhere in the middle.

**Worst Case: O(n)**

* Product is at the last position or is not present.

### For Binary Search

**Best Case: O(1)**

* Product is found at the middle.

**Average Case: O(log n)**

**Worst Case: O(log n)**

Binary Search works only on sorted data.

## Which Is suitable for the platform?

=>Binary search is suitable for the Ecommerce platform because it is faster for large datasets.Just the products need to be sorted before searching.

Linear search is easy to implement but slower for large datasets.
