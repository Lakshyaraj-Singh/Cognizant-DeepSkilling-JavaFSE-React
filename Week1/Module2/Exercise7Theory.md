# Exercise 7: Financial Forecasting

## Explain the concept of recursion and how it can simplify certain problems

=> Recursion is a method where function calls itself until a base condition is reached.It breaks a big problem into smaller similar problems and make the code simple.

## Time Complexity

=> The recursive method makes one recursive call for each year.

Time Complexity: O(n)

Space Complexity: O(n)

## How to optimize the recursive solution?

=> We can use an iterative approach or Dynamic Programming to avoid excessive recursive calls ,reduce memory usage.
