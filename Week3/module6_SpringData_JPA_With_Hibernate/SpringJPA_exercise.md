# Spring Data JPA 

## What is Spring Data JPA?

=> Spring Data JPA is a Spring module which makes database operations easy. It provides ready made methods for CRUD operations so  that we don't need to write SQL queries for every operation,its faster.

## Why use Spring Data JPA?
=> It reduces boilerplate code, saves developmnt time and lets us interact with the database using Repository interfaces instead of writing SQL manually.

## How does Spring Data JPA work?

=> We create an Entity class, a Repository interface and configure the database thenSpring automatically creates the implementation of the Repository and performs the required database operations.
